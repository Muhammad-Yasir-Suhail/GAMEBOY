# GameBoy (SFML + C++)

**Overview**  
A retro-style GameBoy system built in C++ using SFML. It contains multiple classic mini-games inside a single unified interface inspired by old handheld consoles.

**Games Included**  
Wordle – Guess the hidden word using letter feedback  
Hangman – Find the word before the character is fully drawn  
Snake – Grow the snake by eating food without hitting walls or yourself  

**Features**  
- Single menu system for all games  
- Lightweight SFML-based graphics  
- Keyboard-driven controls  
- Classic arcade-style gameplay  
- Simple retro interface  

**Technologies Used**  
- C++  
- SFML (Simple and Fast Multimedia Library)

**Installation**

Clone the repository  
Open the project in your C++ IDE  
(Visual Studio / Code::Blocks / CLion)
Install and link SFML properly  
Build and run the project  


Menu  
- Arrow keys → Navigate  
- Enter → Select  

Snake  
- W/A/S/D or Arrow keys → Move  

Wordle / Hangman  
- Keyboard input → Guess letters/words  


**Project Idea**  
The goal was to combine multiple classic mini-games into a single retro GameBoy-style system with a unified interface and simple controls.
